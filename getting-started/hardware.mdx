---
title: Hardware Requirements
description: Supported hardware configurations for running Inference.net nodes
---

Inference.net supports many different hardware configurations across MacOS, Windows, and Linux. This page provides a list of the most common hardware that Inference.net supports.

<Info>
**Minimum System Requirements**:
- A GPU in the list below with **at least 16GB of VRAM**
- 16GB RAM
- 30GB free disk space
- Reliable 100Mbps+ network bandwidth. Test it here: [Fast.com](https://fast.com/)
</Info>

## Supported Apple Silicon Chips

Inference.net supports a range of Apple Silicon chips, provided the above minimum system requirements are met.


| Chip Series | Variants                   | Notes                                                                 |
|-------------|---------------------------|-----------------------------------------------------------------------|
| M1          | M1, M1 Pro, M1 Max, M1 Ultra | Fully supported                                                       |
| M2          | M2, M2 Pro, M2 Max, M2 Ultra | Fully supported                                                       |
| M3          | M3, M3 Pro, M3 Max          | Fully supported (latest generation as of 2025)                        |
| M4          | M4, M4 Pro, M4 Max          | Fully supported (latest generation released in 2024)                  |
